# Guides Panel: Featured-Article Pattern

## Problem
The Guides panel reads as a flat list. Every item carries equal weight, so nothing signals where a confused PPO shopper should begin. The result is choice paralysis and weak topical signals to search engines.

## Recommendation: feature one cornerstone

Feature **In-network vs out-of-network costs** (`/guides/in-network-vs-out-of-network-costs.html`).

**Why this guide.** It answers the single highest-demand question a PPO shopper asks before booking, and it sits high in the funnel: someone comparing in-network versus out-of-network is actively deciding, not just browsing. It links naturally to plan comparison and to verify-coverage actions, so it routes traffic toward conversion rather than dead-ending on definitions. The glossary is a strong supporting page but serves reference lookups, not decisions, so it belongs in the grouped list.

## How the featured item should look

Give it a larger card at the top of the panel, separated from the list by spacing and a quiet background tint (use `--cream-card`). No icons, no arrows, no decorative flourishes. Distinction comes from size, a Fraunces heading, and a one-line plain summary.

Featured copy:
> **In-network vs out-of-network costs**
> What your PPO actually pays at each office, and why the same crown can cost two different prices.

## Grouping the rest

Two labeled groups beneath the featured card:

**How PPO works**
- PPO dental glossary (`/guides/glossary/`)
- Medi-Cal vs PPO dental (`/guides/medi-cal-vs-ppo-dental.html`)

**Treatment costs**
- Root canal cost (`/guides/root-canal-cost.html`)
- Crown cost (`/guides/crowns/cost.html`)

Use descriptive anchor text exactly as the page topic, never generic labels.

## Topical-authority benefit

Surfacing these cornerstone guides in the every-page nav gives each one a sitewide internal link, concentrating link equity on the pages we most want to rank. Consistent descriptive anchors reinforce the topic of each target, and clustering them by theme builds a clear hub structure that signals subject-matter depth across the PPO dental category.
