# 03 — Content outline: covercapy-network-effect.html

Voice: thought-leadership, editorial, confident but honest. Explain mechanisms, not metrics.
No em-dashes, no roman numerals. Frame as "how it is designed to compound," never as proven KPIs.

## Section order

1. HERO
   - Eyebrow: "CoverCapy / The Network Effect"
   - H1: "Why the CoverCapy network gets better as it grows."
   - Lead: most directories add listings. CoverCapy compounds. Each verified office and prepared
     patient improves the next match.
   - TL;DR quick-answer block (GEO).
   - CTAs: Explore the dentist portal / Find an in-network dentist.

2. FIRST PRINCIPLE — network effects vs a directory
   - Define network effect (demand-side; each participant raises utility for all). Cite Stripe
     Atlas / Cobbleweb. Contrast a static directory (linear) with a flywheel (compounding).
   - Distinguish flywheel (supply/operational compounding) from network effect (demand-side);
     a network effect is one loop inside the larger flywheel. Cite Metavert / Think Insights.

3. THE FIVE LOOPS (the core editorial spine, one block each)
   Loop 1 — Eligibility verification ("Plaid for dental eligibility")
     - Verify the exact PPO network before the visit. Removes the "will they take my plan"
       anxiety. Member ID never stored. Each verification is a clean demand signal.
   Loop 2 — The first dentist after coverage
     - The office matched right after coverage tends to become the patient's dentist. Owning
       that verified, in-network, modern moment is durable. One event becomes a relationship.
   Loop 3 — Patients recruit dentists
     - Nomination loop: demand pulls supply. Offices arrive invited by patients who trust them,
       not via cold sales. Cheaper, warmer supply growth (contrast ~$286/lead acquisition cost).
   Loop 4 — Gamified participation (Capy Crowns / Diamonds)
     - Participation, completeness, accreditation, trust indicators feed recognition + ranking.
       Supply-side engagement up -> demand-side quality up. No promised position or volume.
   Loop 5 — The accreditation quality ratchet
     - Capy Accreditation reviews offices to a standard. Higher average quality raises trust,
       attracting patients, making the network more valuable to join. Reviewed, labeled.

4. HOW THE LOOPS INTERLOCK (the flywheel diagram section)
   - Editorial synthesis: verification feeds matching; first-dentist converts; happy patients
     nominate; gamification + accreditation raise quality; quality attracts patients; repeat.
   - Include a simple, accessible CSS/SVG flywheel/loop visual (5 nodes, scoped, decorative).

5. THE DATA LAYER
   - Layered data network effect: verifications and matches make recommendations sharper over
     time. The network is the product. Cite Modern Data 101. Frame as designed direction.

6. WHY THIS IS A MOAT
   - Two-sided + data effects are hard to copy: a new entrant starts with no verified offices
     and no prepared patients, so its first match is worse than CoverCapy's millionth.
   - Honesty line: the moat is earned loop by loop, not assumed.

7. WHAT IT MEANS FOR YOU
   - For patients: better matches, real verification, free tools. Link compare + find.
   - For dentists: warm demand, a clean verified profile, a fair quality ladder. Link portal,
     membership, PPO patient acquisition, Platinum Elite.

8. FINAL CTA + bridge to /about.html.

9. Page legal note (scoped, truthful: not an insurer/practice; placement reviewed and labeled;
   mechanisms not guarantees).
